The LFO's delay slider describes the period of time between triggering the LFO, and when the LFO starts to become audible.

| Slider | Delay (before anything is audible) | Attack duration (time between 0 and 1) |
|--------|------------------------------------|----------------------------------------|
| 0 | 0 | 0 |
| 2.5 | 0.0639 | 0.053 |
| 5 | 0.85 | 0.188 |
| 7.5 | 1.2 | 0.348 |
| 10 | 2.786 | 1 |
